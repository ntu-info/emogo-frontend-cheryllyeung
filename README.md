# EmoGo - Emotion Tracking App

An interactive emotion tracking mobile application built with React Native and Expo Router.

- **Intensity Rating**: Rate your emotion intensity from 1-10
- **Notes**: Add optional notes to your emotion records
- **History**: View all your recorded emotions in a timeline
- **Statistics**: See your most common emotions and total records
- **Data Export**: Export your emotion data in JSON format
- **Persistent Storage**: All data saved locally using AsyncStorage

## How to run

1. Install dependencies:

   ```bash
   npm install
   # or
   yarn
   ```

2. Start the dev server:

   ```bash
   npx expo start --tunnel
   ```

3. Open the app on a device or emulator using the Expo dev tools.
